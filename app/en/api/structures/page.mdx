## What these are

Structures aren't modules you call — they're **object type definitions**. When a method asks for options or hands you data (a `Cookie`, a `Display`, a `Rectangle`), this section tells you exactly which fields exist and what each one means.

<EasyNote>
If modules are the verbs (do something), structures are the nouns (the shapes of the things). You don't "run" a `Cookie` — you read or build one.
</EasyNote>
