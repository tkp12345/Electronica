## The five tabs you'll live in

| Tab | What for |
|---|---|
| **Elements** | DOM inspection, live CSS edits |
| **Console** | Logs, expressions, error stacks |
| **Network** | API and file-load traces |
| **Performance** | Frame and CPU profile |
| **Memory** | Leak hunting, heap snapshots |

## Open it automatically

```js filename="main.js"
if (process.env.NODE_ENV === 'development') {
  win.webContents.openDevTools({ mode: 'detach' })
}
```

<Callout type="info" emoji="🧒">
`mode: 'detach'` opens DevTools in its own window so it doesn't cover a small app window.
</Callout>

## Bind your own shortcut

```js filename="main.js"
win.webContents.on('before-input-event', (event, input) => {
  if (input.key === 'F12') win.webContents.toggleDevTools()
})
```

## Block it in production

```js
if (process.env.NODE_ENV === 'production') {
  win.webContents.on('devtools-opened', () => win.webContents.closeDevTools())
}
```
