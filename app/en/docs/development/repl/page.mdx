## How to start it

```bash filename="terminal"
electron --interactive
```

Or, in code:

```js filename="main.js"
require('node:repl').start({ prompt: 'electron> ' })
```

## When it shines

- Spot-check a new API in seconds (`require('electron').screen.getAllDisplays()`).
- Inspect live variables mid-debug.
- Get the call shape right _before_ you write the file.

When the Node attach debugger is connected, the _Debug Console_ evaluates expressions on the live process. A lightweight REPL alternative.
