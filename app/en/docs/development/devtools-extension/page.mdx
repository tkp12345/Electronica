## One-line load

```js filename="main.js"
const { app, session } = require('electron')
const path = require('node:path')

app.whenReady().then(async () => {
  // Point at the unpacked extension folder
  await session.defaultSession.loadExtension(
    path.join(__dirname, 'extensions', 'react-devtools'),
  )
})
```

<EasyNote>
You point at an _unpacked folder_, not the Chrome Web Store. That makes it work offline and in locked-down corporate environments.
</EasyNote>

## Good combos

- React component tree → **React Developer Tools**
- Redux/RTK state → **Redux DevTools**
- Network → the built-in DevTools _Network_ tab

## Don't ship them in production

```js
if (process.env.NODE_ENV === 'development') {
  await session.defaultSession.loadExtension(/* ... */)
}
```
