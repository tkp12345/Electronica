## 필요한 것 (체크리스트)

<Steps>

### Node.js 20 이상
Electron 은 Node.js 위에서 돕니다. 터미널에서 버전을 확인하세요.

```bash filename="terminal"
node -v   # v20.x 이상이면 OK
```

<Callout type="info" emoji="🧒">
Node.js 는 _자바스크립트를 브라우저 밖에서 돌릴 수 있게 해 주는 엔진_ 이에요. Electron 의 "Main 프로세스" 는 사실 Node.js 프로그램입니다.
</Callout>

### npm (Node.js 와 함께 자동 설치)
패키지(외부 코드 묶음)를 받아 오는 도구. `npm -v` 로 동작 확인.

### 명령줄(터미널)에 익숙해지기
`cd`, `ls`, `mkdir`, `npm install` 정도면 충분합니다.

### 코드 편집기
VS Code 가 가장 친절합니다 (JS/TS 지원 + Electron 디버깅 확장 풍부).

</Steps>

## Git 도 같이 깔아두면 좋아요

소스 관리 도구. 튜토리얼은 git 없이도 가능하지만, 실제 앱을 만들 땐 사실상 필수예요.

```bash filename="terminal"
git --version
```

## OS 별 한 가지씩 더

<Tabs items={['macOS', 'Windows', 'Linux']}>
  <Tabs.Tab>
    Xcode Command Line Tools — `xcode-select --install` 한 번이면 됩니다.
  </Tabs.Tab>
  <Tabs.Tab>
    [Windows Build Tools](https://github.com/felixrieseberg/windows-build-tools) 는 더 이상 자동 설치되지 않아요. 네이티브 모듈을 쓸 일이 생기면 그때 깔면 됩니다.
  </Tabs.Tab>
  <Tabs.Tab>
    배포판마다 다르지만 보통 `build-essential`(데비안 계열), `base-devel`(아치 계열) 정도가 추가로 필요해요.
  </Tabs.Tab>
</Tabs>
