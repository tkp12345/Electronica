## 폴더 만들기

<Steps>

### 새 프로젝트 폴더
```bash filename="terminal"
mkdir my-electron-app && cd my-electron-app
npm init -y
```

`npm init -y` 는 _"질문 없이 기본값으로 package.json 만들어라"_ 라는 뜻이에요.

### Electron 설치

<Tabs items={['npm', 'pnpm', 'yarn']}>
  <Tabs.Tab>
    ```bash filename="terminal"
    npm install --save-dev electron
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash filename="terminal"
    pnpm add -D electron
    ```
  </Tabs.Tab>
  <Tabs.Tab>
    ```bash filename="terminal"
    yarn add -D electron
    ```
  </Tabs.Tab>
</Tabs>

<EasyNote>
`--save-dev` 는 _"이 패키지는 개발할 때만 필요하지, 배포된 앱에는 안 들어가도 돼"_ 표시예요. Electron 자체는 빌드 도구라 dev 분류가 자연스럽습니다.
</EasyNote>

### `main.js` 만들기 — 앱의 시작점

```js filename="main.js"
const { app, BrowserWindow } = require('electron')

function createWindow() {
  const win = new BrowserWindow({ width: 800, height: 600 })
  win.loadFile('index.html')
}

// 모든 창과 모든 모듈이 준비된 순간
app.whenReady().then(createWindow)

// 맥에서는 dock 아이콘 클릭으로 다시 창 열기
app.on('activate', () => {
  if (BrowserWindow.getAllWindows().length === 0) createWindow()
})

// 윈도우/리눅스: 마지막 창이 닫히면 앱 종료
app.on('window-all-closed', () => {
  if (process.platform !== 'darwin') app.quit()
})
```

<EasyNote>
"앱 준비됨 → 창 생성" 한 줄이 본체고, 나머지 두 개는 _맥/윈도우의 창 닫기 관습 차이를 맞춰주는 보너스_ 입니다.
</EasyNote>

### `index.html` — 실제로 보일 내용

```html filename="index.html"
<!DOCTYPE html>
<html>
  <body>
    <h1>Hello, Electron 👋</h1>
  </body>
</html>
```

### `package.json` 수정 — 진입점과 실행 스크립트

```json filename="package.json"
{
  "name": "my-electron-app",
  "version": "1.0.0",
  "main": "main.js",
  "scripts": {
    "start": "electron ."
  }
}
```

### 실행

```bash filename="terminal"
npm start
```

작은 창이 열리고 "Hello, Electron 👋" 가 보이면 성공!

</Steps>

<Callout type="warning">
창이 안 뜨면: ① `electron .` 가 설치 폴더 안에서 실행됐는지, ② `main.js` 가 package.json 의 `main` 과 같은 이름인지 확인하세요.
</Callout>

## 쉽게 풀어 설명

이번 튜토리얼은 _가장 작은 Electron 앱_ 한 채를 짓는 일이에요. 빈 폴더를 만들고, 그 안에 세 개의 파일만 두면 됩니다. `main.js` 는 _앱이 켜질 때 가장 먼저 읽히는 설계도_, `index.html` 은 _창 안에 보일 내용_, `package.json` 은 _어디서부터 시작할지 알려주는 명함_ 이라고 생각하면 돼요.

코드의 핵심은 단 두 줄이에요. `app.whenReady().then(() => ...)` 가 "Electron 이 준비되면 안에 적은 일을 해라" 라는 뜻이고, 그 안의 `new BrowserWindow({...}).loadFile('index.html')` 이 "가로 800·세로 600 짜리 창을 만들고 우리 HTML 을 띄워라" 라는 뜻이에요. 나머지 코드는 _맥에선 dock 아이콘을 다시 누르면 창이 열리고, 윈도우/리눅스에선 마지막 창이 닫히면 앱이 종료된다_ 는 OS 별 관습을 맞춰주는 _작은 매너_ 일 뿐입니다.

마지막에 `npm start` 한 줄로 실제 창이 뜨면, 사실상 _앱의 절반_ 은 다 만든 거예요. 다음 페이지부터는 이 작은 창에 _안전한 방식으로 기능_ 을 붙여 나가는 이야기로 들어갑니다.
