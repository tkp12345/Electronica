## 1) 애플리케이션 메뉴

```js filename="main.js"
const { app, Menu } = require('electron')

const template = [
  {
    label: '파일',
    submenu: [
      { label: '새로 만들기', accelerator: 'CmdOrCtrl+N', click: () => console.log('new') },
      { type: 'separator' },
      { role: 'quit' },
    ],
  },
]

Menu.setApplicationMenu(Menu.buildFromTemplate(template))
```

<EasyNote>
`accelerator: 'CmdOrCtrl+N'` 한 줄이 _"맥에서는 ⌘N, 윈도우/리눅스에서는 Ctrl+N"_ 을 자동으로 처리해 줍니다.
</EasyNote>

## 2) 전역 단축키 (앱이 포커스 없어도 동작)

```js filename="main.js"
const { app, globalShortcut } = require('electron')

app.whenReady().then(() => {
  globalShortcut.register('CommandOrControl+Shift+P', () => {
    console.log('Palette!')
  })
})

app.on('will-quit', () => globalShortcut.unregisterAll())
```

## 3) 대화상자(Dialog)

```js filename="main.js"
const { dialog } = require('electron')

dialog.showOpenDialog({
  properties: ['openFile'],
  filters: [{ name: 'Images', extensions: ['png', 'jpg'] }],
}).then(({ filePaths }) => console.log(filePaths))
```

## 4) 시스템 알림

```js filename="renderer or preload"
new Notification('빌드 완료', { body: '5.2초 걸렸어요' })
```

<EasyNote>
브라우저의 Notification API 와 동일한 모양 — Electron 이 그대로 OS 알림 센터에 연결합니다.
</EasyNote>

## 5) 트레이 아이콘

```js filename="main.js"
const { app, Tray, Menu } = require('electron')
const path = require('node:path')

let tray
app.whenReady().then(() => {
  tray = new Tray(path.join(__dirname, 'icon.png'))
  tray.setToolTip('My App')
  tray.setContextMenu(Menu.buildFromTemplate([{ role: 'quit' }]))
})
```

## 쉽게 풀어 설명

지금까지의 우리 앱은 _창 하나에 "Hello World" 만 떠 있는_ 단순한 상태였어요. 하지만 사용자가 "이거 진짜 앱이네" 라고 느끼는 순간은 _데스크톱답게 동작할 때_ 예요. 상단 메뉴바에 "파일 → 새로 만들기" 가 보이고, ⌘N 단축키가 먹히고, "파일 열기" 를 누르니 OS 파일 선택 창이 뜨고, 백그라운드에서 일하면 알림으로 알려 주는 것들 말이에요.

이 페이지의 다섯 도구(메뉴·전역 단축키·대화상자·알림·트레이) 가 그 _데스크톱 스러움_ 을 만들어 주는 기본 부품이에요. 신기한 건 코드가 의외로 짧다는 점. 메뉴는 _배열_, 단축키는 _문자열_, 알림은 _Notification 한 줄_ 이면 끝. Electron 이 "OS 별 차이" 를 알아서 흡수해 주니 우리는 _하고 싶은 일만_ 적으면 돼요.

특히 `'CmdOrCtrl+N'` 같은 단축키 표현이 마음에 들 거예요. 같은 문자열이 맥에선 ⌘N, 윈도우/리눅스에선 Ctrl+N 으로 알아서 변신해요. 다음 [패키징](/ko/docs/tutorial/packaging) 페이지에서 친구 컴퓨터로도 보낼 수 있는 진짜 설치 파일을 만들어 봅니다.
