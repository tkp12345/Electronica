## 추천 3종

| 이름 | 특징 |
|---|---|
| **electron-quick-start** | 가장 작은 시작점, 패턴 익히기 좋음 |
| **electron-forge** + 템플릿 (`vite`, `webpack-typescript`) | 공식 풀스택 도구체인 |
| **electron-react-boilerplate** | React + TypeScript + Redux/RTK + Jest 까지 세팅 |

## Forge 템플릿으로 시작

```bash filename="terminal"
npx create-electron-app@latest my-app --template=vite-typescript
```

<Callout type="info" emoji="🧒">
`--template=` 한 줄이 _"빈 종이 대신 격자 노트로 시작"_ 이라는 뜻이에요. Vite 의 빠른 HMR 까지 자동 세팅.
</Callout>
