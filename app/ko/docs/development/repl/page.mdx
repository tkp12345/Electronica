## 켜는 법

```bash filename="terminal"
electron --interactive
```

또는 코드 안에서:

```js filename="main.js"
require('node:repl').start({ prompt: 'electron> ' })
```

## 어디에 좋나

- 새 API 동작을 5초 안에 확인 (`require('electron').screen.getAllDisplays()`)
- 디버깅 중 즉석 변수 확인
- 코드를 작성하기 _전_ 에 _어떻게 부르는지_ 익히기
