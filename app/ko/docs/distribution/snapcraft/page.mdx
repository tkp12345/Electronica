## Snap 의 특징

- **Universal Linux** — 배포판에 상관없이 같은 `.snap` 파일
- **자동 업데이트** — Snap Store 가 백그라운드에서 알아서 갱신
- **샌드박스(AppArmor 기반)** — 권한이 명시적
- **Stable / Candidate / Beta / Edge** 4채널 관리

## Forge maker

```js filename="forge.config.js"
{
  name: '@electron-forge/maker-snap',
  config: {
    summary: 'My cool app',
    description: 'Cooler than yours',
  },
}
```

<EasyNote>
".deb 은 _배포판 동네 가게_, .snap 은 _전국 체인_". 어디서나 같은 모양으로 동작하지만, 컨테이너라 약간 무거워요.
</EasyNote>

## 업로드

```bash filename="terminal"
snapcraft login
snapcraft upload my-app_1.0.0_amd64.snap --release=stable
```

## 다른 옵션과 비교

- `.deb` / `.rpm` — 가벼움, 배포판 특화
- `.AppImage` — 설치 불필요, 단일 파일
- **Snap** — 자동 업데이트가 큰 장점
