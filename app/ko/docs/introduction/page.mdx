## 한 줄로 말하면

> **"내가 만든 웹사이트를 더블클릭하는 앱으로 바꿔주는 포장지"**

웹에서 쓰던 그대로의 HTML/CSS/JS 코드를, Chromium(크롬 안에 들어 있는 브라우저 엔진)으로 그리고, Node.js(서버에서 자바스크립트를 돌리는 도구)로 컴퓨터의 파일·네트워크·창 같은 기능을 다룰 수 있게 묶어 줍니다.

<Callout type="info" emoji="💡">
**비유**: 웹사이트가 _그림_ 이라면, Electron 은 _액자 + 받침대 + 콘센트_ 를 달아 벽이 아닌 책상 위에 세워 놓을 수 있게 해 주는 셋트예요.
</Callout>

## 어떤 앱이 Electron으로 만들어졌을까?

VS Code, Slack, Discord, Figma 데스크톱, Notion, Postman 같은 친숙한 앱들이 모두 Electron 기반입니다. "내가 쓰는 그 앱"이 사실은 웹사이트였다는 뜻이에요.

## 기본 구성 요소 3가지

Electron 앱은 결국 이 세 개의 조각으로 이루어집니다.

| 조각 | 무엇이냐 | 비유 |
|---|---|---|
| **Chromium** | 화면을 그려 주는 브라우저 엔진 | 영사기 |
| **Node.js** | 파일·네트워크 같은 OS 기능에 접근 | 무대 뒤 기술팀 |
| **Electron API** | 창·메뉴·트레이·자동업데이트 등 데스크톱 전용 기능 | 무대 위 조명·소품 |

## 가장 작은 앱의 모양

```js filename="main.js"
// Electron 의 진입점 — 이 파일이 가장 먼저 실행된다
const { app, BrowserWindow } = require('electron')

app.whenReady().then(() => {
  // 화면에 보이는 창(window) 하나 생성
  const win = new BrowserWindow({ width: 800, height: 600 })
  // 그 창에 띄울 내용 — 우리가 만든 HTML 파일
  win.loadFile('index.html')
})
```

<Callout type="info" emoji="🧒">
위 코드는 사실상 _"앱이 준비되면 → 가로 800·세로 600 짜리 창을 열고 → index.html 을 띄워라"_ 한 줄짜리 요리법이에요. Electron 의 핵심은 결국 이 패턴의 확장입니다.
</Callout>

```html filename="index.html"
<!DOCTYPE html>
<html>
  <body>
    <h1>Hello, Electron!</h1>
  </body>
</html>
```
