## 무엇인가요

Structures 는 호출하는 모듈이 아니라 **객체 타입 정의** 예요. 어떤 메서드가 옵션을 요구하거나 데이터(`Cookie`, `Display`, `Rectangle` 등)를 돌려줄 때, 이 섹션이 정확히 어떤 필드가 있고 각각이 무슨 뜻인지 알려줘요.

<EasyNote>
모듈이 동사(무언가를 한다)라면, 구조체는 명사(그 무언가의 모양)예요. `Cookie` 를 "실행"하지는 않아요 — 읽거나 만들 뿐이죠.
</EasyNote>
