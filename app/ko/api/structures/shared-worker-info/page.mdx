`SharedWorkerInfo` 객체는 하나의 공유 워커에 대한 기본 정보를 담고 있습니다. `app` 모듈에서 (예를 들어 세션이나 `ServiceWorkers` 인스턴스에 속한 공유 워커 목록을 조회할 때) 이 객체를 돌려받기 때문에, 현재 무엇이 실행 중인지 확인할 수 있습니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `id` | `string` | 공유 워커의 고유 id입니다. |
| `url` | `string` | 공유 워커의 url입니다. |

```js
const { app } = require('electron')

// 돌려받게 되는 형태의 예시
const info = {
  id: '1',
  url: 'https://example.com/worker.js'
}
```

<EasyNote>
백그라운드 도우미에 붙은 이름표 같은 것입니다. 단지 id와 코드가 있는 위치(url)만 담겨 있습니다.
</EasyNote>

## 쉽게 말하면

공유 워커는 여러 페이지가 동시에 대화할 수 있는 작은 백그라운드 스크립트입니다. Electron에 어떤 공유 워커가 있는지 물어보면, 각각이 `SharedWorkerInfo`로 돌아와서 그 워커의 고유 id와 코드를 불러온 url을 알 수 있습니다. 앱에서 실행 중인 백그라운드 워커를 살펴보거나, 기록하거나, 디버깅할 때 사용하면 됩니다.
