`WebSource`는 평가(실행)할 소스 코드 조각과, 그 코드의 출처를 나타내는 선택적 URL을 함께 담는 객체입니다. `WebFrameMain.executeJavaScript`나 `webFrame` 모듈처럼 페이지에 코드를 주입하거나 실행하는 API에서 사용됩니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `code` | `string` | 실행하거나 주입할 소스 코드입니다. |
| `url` | `string` | _(선택)_ 코드의 출처를 나타내는 URL입니다. 스택 트레이스나 개발자 도구가 의미 있는 위치를 가리킬 수 있도록 도와줍니다. |

```js
const source = {
  code: 'console.dir(window)',
  url: 'electron://my-injected-script.js',
}
```

<EasyNote>
코드가 적힌 포스트잇 한 장에, "이 코드는 어떤 파일에서 왔어요"라고 알려주는 작은 라벨이 붙어 있는 모습이라고 생각하시면 됩니다.
</EasyNote>

## 쉽게 말하면

Electron에게 페이지 안에서 JavaScript 코드 조각을 실행해 달라고 요청할 때, 그 코드를 건네주는 방법이 바로 `WebSource`입니다. `code`는 실제로 실행하고 싶은 스크립트이고, 선택적인 `url`은 문제가 생겼을 때 오류 메시지나 디버거가 "이 코드는 어느 파일에서 왔는지" 알려줄 수 있도록 붙이는 친숙한 이름일 뿐입니다.
