`FileFilter` 객체는 선택 가능한 파일 종류 하나의 묶음을 설명합니다. 이 객체들의 배열을 `dialog` 모듈의 `showOpenDialog`, `showSaveDialog` 메서드에 (`filters` 옵션으로) 전달하면, 사용자가 고를 수 있는 파일을 제한할 수 있습니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `name` | `string` | 대화상자에 표시되는 이 필터 묶음의 이름입니다(예: "이미지"). |
| `extensions` | `string[]` | 허용할 파일 확장자 목록으로, 앞의 점(.)이나 와일드카드 없이 작성합니다(예: `['jpg', 'png']`). 모든 파일을 허용하려면 `['*']`를 사용합니다. |

```js
const { dialog } = require('electron')

dialog.showOpenDialog({
  filters: [
    { name: '이미지', extensions: ['jpg', 'png', 'gif'] },
    { name: '동영상', extensions: ['mkv', 'avi', 'mp4'] },
    { name: '모든 파일', extensions: ['*'] }
  ]
})
```

<EasyNote>
저장하거나 열기 창 아래쪽에 있는 "파일 형식" 드롭다운과 같습니다. 파일 종류마다 항목 하나씩 들어가는 셈입니다.
</EasyNote>

## 쉽게 풀어 설명

누군가에게 파일을 열거나 저장하게 할 때, 보통 컴퓨터의 모든 파일을 다 보여 주고 싶지는 않습니다. `FileFilter`를 사용하면 "이미지만 보여 줘" 또는 "텍스트 파일만 보여 줘"라고 지정할 수 있고, 각 묶음에 알아보기 쉬운 이름과 확장자 목록을 붙일 수 있습니다. 이런 객체를 작은 배열로 만들어 파일 대화상자에 넘겨 주면 됩니다.
