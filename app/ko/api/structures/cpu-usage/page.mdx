`CPUUsage`는 하나의 프로세스가 사용하는 CPU 사용량을 설명하는 평범한 객체입니다. `app.getCPUUsage()`가 반환하는 값이며, 프로세스가 얼마나 열심히 일하고 있는지 측정할 수 있게 해줍니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `percentCPUUsage` | `number` | 마지막으로 `getCPUUsage`를 호출한 이후 사용된 CPU의 백분율입니다. 첫 번째 호출은 0을 반환합니다. |
| `cumulativeCPUUsage` | `number` | _(선택)_ 프로세스가 시작된 이후 사용한 총 CPU 시간(초)입니다. |
| `idleWakeupsPerSecond` | `number` | 마지막으로 `getCPUUsage`를 호출한 이후 초당 평균 유휴 CPU 깨우기 횟수입니다. 첫 번째 호출은 0을 반환합니다. Windows에서는 항상 0을 반환합니다. |

```js
const { app } = require('electron')

// 몇 초마다 CPU 사용량을 확인합니다
setInterval(() => {
  const usage = app.getCPUUsage()
  // { percentCPUUsage: 3.2, cumulativeCPUUsage: 12.4, idleWakeupsPerSecond: 0 }
  console.error(usage.percentCPUUsage)
}, 5000)
```

## 쉽게 풀어 설명

`CPUUsage`는 내 앱이 사용자의 컴퓨터에서 얌전하게 동작하고 있는지 확인하고 싶을 때 사용합니다. `percentCPUUsage`가 계속 올라간다면 어딘가에서 무한 루프에 빠졌거나 너무 많은 일을 하고 있을 수 있습니다. 이 수치를 시간에 따라 기록해 두면 성능 문제를 발견하고, 무거운 작업을 메인 프로세스 밖으로 옮길 시점을 판단하는 데 도움이 됩니다.
