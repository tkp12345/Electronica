`ThumbarButton`은 Windows에서 작업 표시줄 아이콘 위에 마우스를 올렸을 때 나타나는 썸네일 툴바에 표시되는 버튼 하나를 나타냅니다. 이 객체들의 배열을 `win.setThumbarButtons(buttons)`에 전달해서 사용합니다. Windows 전용 구조체입니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `icon` | `NativeImage` | _(Windows)_ 썸네일 툴바에서 버튼에 표시되는 아이콘입니다. |
| `click` | `Function` | _(Windows)_ 사용자가 버튼을 클릭했을 때 호출되는 콜백 함수입니다. |
| `tooltip` | `string` | _(선택)_ _(Windows)_ 버튼의 툴팁에 표시되는 텍스트입니다. |
| `flags` | `string[]` | _(선택)_ _(Windows)_ 버튼의 상태와 동작을 제어합니다. 기본값은 `['enabled']`입니다. 사용 가능한 값: `enabled`, `disabled`, `dismissonclick`, `nobackground`, `hidden`, `noninteractive`. |

```js
const { nativeImage } = require('electron')

const button = {
  tooltip: 'Play',
  icon: nativeImage.createFromPath('/path/to/play.png'),
  flags: ['enabled'],
  click() {
    console.info('clicked')
  },
}
```

<EasyNote>
ThumbarButton은 음악 앱 작업 표시줄 미리보기에 붙은 작은 재생/일시정지 버튼 같은 것입니다. 창을 열지 않고도 바로 누를 수 있는 컨트롤이죠.
</EasyNote>

## 쉽게 풀어 설명

Windows 작업 표시줄에서 앱 아이콘 위에 마우스를 올리면 작은 미리보기가 떠요. ThumbarButton을 사용하면 그 미리보기 아래에 최대 7개의 작은 컨트롤 버튼을 추가할 수 있는데, 이전 곡, 재생, 다음 곡 같은 미디어 컨트롤에 딱 좋습니다. 각 버튼에는 `icon`과 `click` 핸들러가 필요하고, 나머지는 선택 사항입니다.
