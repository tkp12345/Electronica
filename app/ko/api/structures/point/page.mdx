`Point`는 `x`와 `y` 좌표를 담은 간단한 객체입니다. 위치가 필요한 곳이라면 Electron 어디에서나 등장합니다. 예를 들어 `screen.getCursorScreenPoint()`는 `Point`를 반환하고, `BrowserWindow.setPosition()` 같은 메서드도 같은 형태의 값을 사용합니다.

## 필드

| 필드 | 타입 | 설명 |
|---|---|---|
| `x` | `number` | x(가로) 좌표 값입니다. |
| `y` | `number` | y(세로) 좌표 값입니다. |

<Callout type="info">
`x`와 `y`는 반드시 정수여야 합니다. `Point`를 Electron API에 전달하면 값이 가장 가까운 정수로 자동 반올림됩니다.
</Callout>

```js
const point = { x: 100, y: 250 }
```

<EasyNote>
`Point`는 모눈종이 위의 주소와 같습니다. 한 숫자는 가로로 얼마나 떨어져 있는지, 다른 숫자는 세로로 얼마나 내려가 있는지를 알려줍니다.
</EasyNote>

## 쉽게 풀어 설명

화면을 격자판이라고 생각해 보세요. `x`는 어떤 것이 오른쪽으로 얼마나 떨어져 있는지, `y`는 아래로 얼마나 내려가 있는지를 나타냅니다. `Point`는 이 두 숫자를 하나로 묶어 주어서, Electron이 화면의 어느 지점을 말하는지 정확히 알 수 있게 합니다. 마우스 커서 아래 위치나 창의 모서리 같은 지점을 가리킬 때 쓰입니다.
