# 시작하기

Electron은 **Chromium**과 **Node.js**를 묶어 데스크톱 앱을 만들 수 있게 해주는 프레임워크입니다.

<Callout type="info">
  이 문서는 Roberto의 학습 노트입니다. Electron의 공식 문서를 보완하는 **개인 메모**로 작성됩니다.
</Callout>

## Electron이란?

- **Main 프로세스**: Node.js 환경에서 실행되며 OS 레벨 API에 접근. 앱의 진입점.
- **Renderer 프로세스**: Chromium에서 실행되며 웹페이지처럼 동작. UI 담당.
- 두 프로세스는 **IPC(Inter-Process Communication)** 로 통신합니다.

## 학습 흐름

<Steps>
### 환경 세팅

Node.js 20+ 설치 후, 새 프로젝트에 Electron 의존성을 추가합니다.

### 메인 프로세스 작성

`main.js`에서 `BrowserWindow`를 만들어 첫 윈도우를 띄웁니다.

### Preload 스크립트로 보안 경계 설정

`contextIsolation: true`와 `preload` 조합으로 안전하게 IPC 노출.

### IPC로 메인 ↔ 렌더러 통신

`ipcMain.handle` + `ipcRenderer.invoke` 패턴이 가장 일반적.

### 패키징/배포

`electron-builder` 또는 `electron-forge`로 OS별 인스톨러를 만듭니다.
</Steps>
